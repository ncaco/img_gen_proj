# 카드 생성기

## 프로젝트 개요
메인 캐릭터와 설명을 기반으로 AI가 생성한 요소들(배경, 카드명, 속성, 스킬, 등급 등)을 자동으로 배치하여 트레이딩 카드 게임 스타일의 카드를 생성하는 서비스입니다.

## 카드 구조 정의

### 카드 레이아웃

```
┌─────────────────────────────────────────┐
│  [배경 이미지 - Layer 2 (전체 영역)]     │  ← 배경이 카드 전체를 덮음
│  ┌─────────────────────────────────┐   │
│  │                                 │   │
│  │  ⭕[타입]  [등급]  카드명  [속성]⭕ │  ← 상단 헤더 (투명 배경 오버레이)
│  │  ─────────────────────────────  │   │
│  │                                 │   │
│  │    [메인 캐릭터 이미지 - Layer 1]  │  ← 이미지 영역
│  │                                 │   │
│  │  ─────────────────────────────  │   │
│  │  [스킬 1] (투명 배경 오버레이)     │  ← 스킬 영역
│  │  • 스킬명: 효과 설명              │   │
│  │                                 │   │
│  │  [스킬 2] (투명 배경 오버레이)     │
│  │  • 스킬명: 효과 설명              │   │
│  │                                 │   │
│  │  ─────────────────────────────  │   │
│  │  [설명/플레이버 텍스트]            │  ← 하단 설명 (투명 배경 오버레이)
│  │                                 │   │
│  │  [카드번호] [시리즈] (투명 오버레이)│  ← 하단 메타 정보
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 카드 요소 상세 정의

#### 1. 상단 헤더 영역
- **등급 아이콘**: 카드의 희귀도 표시 (일반, 레어, 슈퍼레어, 울트라레어 등)
- **카드명**: 캐릭터 또는 카드의 이름

#### 2. 속성/타입 영역
- **타입 (왼쪽 상단 동그라미)**: 공격형, 방어형, 지원형 등 - 왼쪽 상단에 동그라미 형태로 표시
- **속성 (오른쪽 상단 동그라미)**: 불, 물, 땅, 바람, 빛, 어둠 등 - 오른쪽 상단에 동그라미 형태로 표시

#### 3. 이미지 영역 (Layer 구조)
- **Layer 2 (배경)**: 카드 전체를 덮는 배경 이미지 (최하단 레이어)
- **Layer 1 (메인 캐릭터)**: 배경 위에 중앙에 배치되는 메인 캐릭터 이미지
- **텍스트 오버레이 레이어**: 모든 텍스트 요소들(카드명, 스킬, 설명, 카드번호 등)은 투명도가 높은 배경 위에 표시되어 배경 이미지 위에 오버레이로 출력됨

#### 4. 스킬 영역 (투명 배경 오버레이)
- **스킬 1**: 첫 번째 능력/스킬
  - 스킬명
  - 효과 설명
  - 필요 코스트 (선택사항)
  - 투명도가 높은 배경 위에 표시되어 배경 이미지 위에 오버레이로 출력
- **스킬 2**: 두 번째 능력/스킬
  - 스킬명
  - 효과 설명
  - 필요 코스트 (선택사항)
  - 투명도가 높은 배경 위에 표시되어 배경 이미지 위에 오버레이로 출력

#### 5. 하단 설명 영역 (투명 배경 오버레이)
- **플레이버 텍스트**: 카드의 배경 스토리나 캐릭터 설명
- 투명도가 높은 배경 위에 표시되어 배경 이미지 위에 오버레이로 출력

#### 6. 메타 정보 영역 (투명 배경 오버레이)
- **카드 번호**: 시리즈 내 카드 번호
- **제작자/시리즈 정보**: 제작자명 또는 시리즈명
- 투명도가 높은 배경 위에 표시되어 배경 이미지 위에 오버레이로 출력

## 제작방식
- **Layer 2 (최하단)**: 배경 이미지 - 카드 전체 영역을 덮음
- **Layer 1**: 메인 캐릭터 이미지 / 태그 - 배경 위에 배치
- **텍스트 오버레이 레이어**: 카드명, 스킬, 설명, 카드번호 등 모든 텍스트 요소는 투명도가 높은 배경 위에 표시되어 배경 이미지 위에 오버레이로 출력
